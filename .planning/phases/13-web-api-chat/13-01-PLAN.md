       - Message 必填（但允许空字符串，由业务层处理），SessionId 可选

       - 逻辑:
         a. 如果 request.Message 为空或仅空白，返回 Results.BadRequest(new { error = "message is required" })

         d. 返回 Results.Ok(new { response = result, sessionId = sessionId })
       - 添加 .WithName("Chat") 和 .WithOpenApi() 用于 Swagger 文档
       - 注意: 返回类型为 JSON（Results.Ok），不是 SSE（per D-03 纯阻塞式响应）

| Threat ID | Category | Component | Disposition | Mitigation Plan |
