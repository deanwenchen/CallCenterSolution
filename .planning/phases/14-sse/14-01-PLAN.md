    1. Get event type name: `var typeName = evt.GetType().Name;`

    3. Return SSE format string: `$"data: {{\"type\":\"{typeName}\",\"data\":{json}}}\n\n"`
    4. This produces: `data: {"type":"WorkflowStarted","data":{...}}\n\n` (per D-14-02, D-14-03)

| HTTP client → /chat/stream | Untrusted JSON body (message, sessionId) crosses here |

| SessionId input → InMemorySessionStore | User-supplied sessionId used as dictionary key — potential key injection |
