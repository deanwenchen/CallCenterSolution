Replace `AuditLogger.cs` TODO stub:

Create `AuditLogEntry` record:
- `Timestamp`, `SessionId`, `ExecutorId`, `InputJson`, `OutputJson`, `EventType`, `PreviousHash`

Create `AuditLogger` class:
- Constructor takes `string auditDirectory = ".audit"`
- `LogAsync(executorId, input, output, eventType, sessionId, ct)`:
  1. Read last line of existing audit file to get previous hash (or empty for first entry)
  2. Serialize entry to JSON
  3. Compute SHA256 of the JSON line → store as `hash`
  4. Append line to `.audit/{sessionId}.jsonl` with SemaphoreSlim for thread safety
- `VerifyChainAsync(sessionId, ct)`:
  1. Read all lines from audit file
  2. For each line: verify `previousHash` matches hash of previous line
  3. Verify each line's `hash` matches SHA256 of content
  4. Return `AuditVerificationResult` (Valid, Invalid, TamperedAt)
- `GetAuditFile(sessionId)`: return full path to audit file
</action>

<acceptance_criteria>
- AuditLogger.LogAsync creates `.audit/{sessionId}.jsonl` with valid JSON
- Each entry has `previousHash` linking to previous entry's hash
- First entry has empty `previousHash`
- AuditLogger.VerifyChainAsync returns Valid for unmodified file
- AuditLogger.VerifyChainAsync detects tampered entry (modified content)
- Build: 0 errors
</acceptance_criteria>

<action>
Replace `SagaBuilder.cs` TODO stub:

Create `SagaCompensationException`:
- Inherits from Exception
- Properties: `FailedStep`, `CompensationStep`, `InnerException`

Create `SagaBuilder` class:
- Fluent API:
  - `OnFailure(string step, Func<CancellationToken, Task> compensation)` — register compensation action
  - `WithRetry(int maxRetries, params TimeSpan[] delays)` — configure retry strategy
  - `ExecuteAsync(Func<CancellationToken, Task> action, ct)` — execute with retry + compensation on failure

Internal logic:
1. Execute action
2. If succeeds → return
3. If fails → check if any OnFailure handlers match the failed step name
4. If match found → execute compensation (also retried with same delays)
5. If compensation fails → throw SagaCompensationException
6. If no match → rethrow original exception

Retry logic:
- On failure, wait for next delay in array
- Retry up to maxRetries times
- If all retries exhausted → proceed to compensation
</action>

<acceptance_criteria>
- SagaBuilder executes action successfully on first try
- SagaBuilder retries on failure (configurable delays)
- SagaBuilder executes compensation when all retries exhausted
- SagaBuilder throws SagaCompensationException if compensation fails
- Build: 0 errors
</acceptance_criteria>
